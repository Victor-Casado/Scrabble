# Bot Starter Exploration

This branch snapshot preserves the evolved `bot-starter` code and tests used
while developing the Lisan/Atreides tournament strategy.

It includes the strategy code, precompute helpers, policy tests, and experiment
support files that were useful during exploration. Token files, build output,
and dependency folders are intentionally excluded from the branch snapshot.

## Commands

```powershell
npm install
npm test
```
